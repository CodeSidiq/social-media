# Social Media Web App

Frontend project for a constitution-driven social media application.

## Core Stack

- Next.js
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Redux Toolkit
- TanStack Query
- Axios
- React Hook Form
- Zod
- Lucide React
- Motion
- clsx
- tailwind-merge
- dayjs

## Project Status

Initial project foundation is ready.

## Available Scripts

- npm run dev
- npm run build
- npm run lint
- npm run start
