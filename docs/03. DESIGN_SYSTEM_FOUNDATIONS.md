========================================
DESIGN SYSTEM
========================================

DESIGN AUTHORITY

This document defines the official design tokens
for the project.

All UI styling must derive from these tokens unless
explicitly approved otherwise.

This design system defines the visual foundation of the UI including:

- Typography
- Color
- Radius
- Spacing

The goal is to ensure visual consistency, scalability, and accessibility
across all components and layouts.
